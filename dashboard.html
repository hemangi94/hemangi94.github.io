This dashboard contains interactive graphics for NYC restaurant inspection data. More details about the grading and scoring process can be found here.

Number of critical violations across restuarants in each borough

rest_inspec %>% 
  filter(
    critical_flag == "Critical"
  ) %>% 
  count(boro) %>% 
  plot_ly(
    x = ~boro, y = ~n,
    type = "bar"
  ) %>% 
  layout(xaxis = list(title = "borough"), 
         yaxis = list(title = "# of critical violations"))
